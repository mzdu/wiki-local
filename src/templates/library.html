API Documentation

Note:

  • Currently the only available return type is JSON
  • All responses contain either 'stat' : 'ok' or 'stat' : 'fail' for error checking.

REST request format

http://wikitheoria.appspot.com/api?method=theMethod¶mName=yourParameter

Methods

getSuggestions

Get a list of suggestions based on a query.
Example: 

http://wikitheoria.appspot.com/api?method=getSuggestions&query=a
Example Response: 
{"query": "a", "suggestions": ["after", "apple", "attitude", "suggestion n"], 'stat' : 'ok'}

Arguments

query (required)
Look for words starting with these letters

getTermDefinitions

Get a list of definitions for a particular term.
Example: 

http://wikitheoria.appspot.com/api?method=getTermDefinitions&term=apple
Example Response: 
	{"definitions": 
		{"pronoun": ["def 1", "def 2", "def n"], 
		"adverb": ["def 1", "def 2", "def n"], 
		"adjective": ["def 1", "def 2", "def n"], 
		"preposition": ["def 1", "def 2", "def n"], 
		"verb": ["def 1", "def 2", "def n"], 
		"nouns": ["def 1", "def 2", "def n"], 
		"conjunction": ["def 1", "def 2", "def n"], 
		"interjection": ["def 1", "def 2", "def n"]
		.
		.
		.}, 
	"term": "apple", 
	'stat' : 'ok'}

Arguments

term (required)
The term to get definitions for.
function (optional) (options = [noun,verb,adjective,pronoun,adverb,preposition,conjunction,interjection,nounPhrase,verbPhrase,prepositionalPhrase,adjectivalPhrase,adverbialPhrase])
Returns only definitions for a particular part of speech
Example: 
http://wikitheoria.appspot.com/api?method=getTermDefinitions&term=apple&function=noun
Example Response: 
{"definitions": {"nouns": ["def 1", "def 2", "def n"]}, "term": "apple", 'stat' : 'ok'}
